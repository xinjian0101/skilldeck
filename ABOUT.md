# About SkillDeck

SkillDeck is a planned local-first desktop application for organizing reusable development workflow files.

## Suggested GitHub About

Local-first workspace for browsing, reviewing, comparing, and packaging reusable development configurations.

## Topics

`developer-tools`, `local-first`, `desktop-app`, `configuration-management`, `typescript`, `tauri`

## Purpose

The project will provide a clear inventory, readable metadata, version comparison, validation reports, and portable exports. Planned behavior will remain separate from released behavior.

## Current status

Foundation phase. The repository currently contains planning and product documentation. No executable release is available.

## Maintenance

Project changes will use focused commits, public roadmap updates, release notes, and documented compatibility tests.
